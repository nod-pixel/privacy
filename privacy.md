# Privacy Policy (Personal Use) — Garmin Data to Splunk
**Effective date:** 2026-03-05

## Who this is for
This is a personal testing project used by a single user (me) to collect my own Garmin activity/health data and visualize it in Splunk dashboards.

## What data I collect
Depending on what I enable, I may collect Garmin data associated with my account, including:
- Activity data (e.g., runs/rides/workouts, duration, distance, pace)
- Wellness/health metrics (e.g., heart rate, sleep summaries, steps, stress/Body Battery)

## Why I collect it
I use the data only to:
- Build and test Splunk dashboards, searches, and alerts
- Explore personal analytics and data quality

## How the data is processed and stored
- Data is retrieved from Garmin (via an API integration) and sent to my private Splunk instance.
- Data is stored only in my Splunk environment and is not published publicly.

## Sharing and disclosure
- I do not sell this data.
- I do not share this data with third parties, except as required to operate my own infrastructure (for example, a hosting provider for my Splunk environment), and only to the extent necessary to provide the service.

## Data retention
- Data is retained in Splunk for testing purposes.
- I may delete data at any time by removing it from Splunk storage and/or deleting relevant indexes.

## Security
I use reasonable security measures for a personal project, such as:
- Authentication and access control on Splunk
- Encryption in transit where available (HTTPS)

## Your choices / revoking access
Because this is for my own data, I can stop collection at any time by:
- Revoking the app’s access in Garmin (or disabling the integration)
- Stopping the collector script/service
- Deleting stored data from Splunk
